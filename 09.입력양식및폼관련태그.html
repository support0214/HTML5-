폼 관련 태그

form태그내에 submit버튼 클릭 시 해당 form안의 작성된
사용자가 입력한 값들을 서버로 전송하면서 요청하는 역할 수행

form action="" method="""

* action속성 : 해당 form에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로 제시
* method속성 : 요청 전송 방식을 지정하는 속성(get(기본값)/post)
 > get방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식
 > post방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식
form내의 submit버튼 클릭시 action 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨(사용자가 입력한 값들을 항상 key=value 세트로 전달)
